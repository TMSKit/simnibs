Requirements
============

Operating Systems
-----------------

Windows
''''''''
SimNIBS has been tested on 64 bit Windows 10


Linux
''''''

SimNIBS has been tested on 64 bit Ubuntu 16.04 and 18.04 as well as CentOS 7.


MacOS
''''''
SimNIBS has been tested on MacOS 10.14 (Mojave) and 10.15 (Catalina)


Hardware
--------

  * 3 GB disk space
  * 8 GB of memory
